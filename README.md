# 🚗 EBAC Motors

Página web desenvolvida para simular o site de uma loja de carros, utilizando HTML, CSS e JavaScript com jQuery e plugins para interação e validação de dados.

## 🚀 Funcionalidades

* Estrutura de site institucional para apresentação de veículos
* Exibição de imagens e ícones no layout
* Interações com JavaScript
* Validação de formulário utilizando jQuery Validate
* Máscara de campos com jQuery Mask

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* jQuery
* jQuery Mask Plugin
* jQuery Validate Plugin

## 📂 Estrutura do Projeto

```bash
EBAC MOTORS/
├── icones/
├── images/
├── index.html
├── main.css
├── jquery/
│   ├── jquery.mask.min.js
│   ├── jquery.validate.min.js
│   ├── messages_pt_BR.min.js
│   └── main.js
```

## ▶️ Como Executar

Abra o arquivo `index.html` no navegador.
